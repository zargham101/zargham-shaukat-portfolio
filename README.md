# Zargham Shaukat - Portfolio Website

Modern, responsive portfolio website showcasing my work as a Senior Software Engineer.

## Tech Stack
- React.js
- Tailwind CSS
- Lucide React Icons

## Local Development
1. Install dependencies: `npm install`
2. Run development server: `npm start`
3. Build for production: `npm run build`

## Deployment
Deployed on Vercel: https://zarghamshaukat.vercel.app
